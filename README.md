# Brainfuck to C parser
A very simple (101 lines) brainfuck to C parser.

## Compilation

Programs needed: a C compiler
Libraries needed: none

Compile with:
```bash
gcc -o bftoc bftoc.c
```

## Usage

Programs needed: a C compiler
Libraries needed: none

Usage:

```bash
./bftoc foo.bf foo.c
gcc -o foo foo.c
./foo
```
